# DID

{% hint style="info" %}
去中心化身份(Decentralized Identity，DID）是结合区块链技术的信任发明，是未来在去中心化社会的身份凭证。 具体来说，它是区块链上的一个地址，由一个人拥有和控制，用来连接与DID 相关的文件，如钱包地址、加密账户、dApp 交互、社交、登录等。
